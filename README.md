## Get started

- To start using the API, you first need to obtain a key by registering for aÂ [Rijksstudio account](https://www.rijksmuseum.nl/en/rijksstudio). You will be given a key instantly upon request, which you can find at the advanced settings of your Rijksstudio account**.**
- Copy your API key to the app_configuration.dart file.
- Run pub get
- Run the app
